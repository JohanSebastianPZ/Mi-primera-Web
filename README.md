# Mi Proyecto: My Life (Panel de Control Personal)

Hola! Soy Johan Martinez. Este es el proyecto en el que he estado trabajando para mi clase de Diseno de Interfaces. La idea era crear una pagina tipo dashboard o panel personal donde pudiera organizar mis cosas diarias (tareas, notas y hasta un podcast) usando HTML, Tailwind CSS y jQuery.

## De que trata el proyecto?
My Life es un sitio web donde puedes gestionar tu perfil de desarrollador. Quise que fuera muy visual, asi que use Tailwind para que se vea moderno y jQuery para que la pagina cobre vida sin tener que recargar el navegador cada vez que haces algo.

---

## Que utilice?
* HTML5: Para toda la estructura.
* Tailwind CSS: Para que los botones, las tarjetas y los textos se vean geniales y para algunas animaciones que hice para el logo.
* jQuery: Mi herramienta principal para que todo sea interactivo (clics, efectos, validaciones).
* FullCalendar: Un plugin externo que agregue para tener un calendario real donde puedo anotar mis entregas.

---

## Cosas divertidas que puedes hacer (Misiones)

Durante el desarrollo me puse varios retos o misiones para practicar:

1. Cambios de look al vuelo: Use codigo para cambiar colores y textos dinamicamente. Por ejemplo, si miras el primer parrafo, veras un mensaje motivador que puse con JS!
2. Lista de tareas: cree un sistema sencillo para agregar tareas a una tabla (usando append) y tambien una opcion para borrar la ultima tarea si te equivocas.
3. Contador de notas: En la seccion de Mis Notas, hice un contador que te dice cuantas letras llevas escritas mientras vas tecleando.
4. Efectos visuales: Al darle al boton de Play en el podcast, el logo desaparece suavemente. Al darle a Edit Profile, la foto de perfil aparece con un efecto de deslizamiento hacia abajo. El menu de arriba (header) se vuelve un poco transparente cuando bajas la pagina (scroll).
5. Mi Agenda: Logre integrar un calendario donde, si haces clic en un dia, te sale un aviso preguntandote que evento quieres guardar.

## Como probarlo
Es super facil:
1. Descarga los archivos o clona el repo.
2. Abre el archivo profile.html en tu navegador.
3. Empieza a jugar con los botones! Prueba agregar tareas o escribir en el cuadro de notas.
